NOTES ON PROGRAMMING:

Inline

Inline codes are codes that deal with the text lines, like (b) or (strong). They change what the lines say. (I'm using parentheses because if I use the actual signs, they won't show up)

Examples of inline are: (b) or (strong) for bold, (em) for italics, (a) and (img)


Block


The other type of programming is Block, which forms an invisible box around the said text. The block htmls I've learned so far are (p),(form), and (div), which all start a new "paragraph". They separate text from other text in different manners
